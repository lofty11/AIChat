# 项目简介

## 下载依赖

### 前端

先进入前端

```
cd front
```

在终端输入如下命令下载依赖

```
npm install
```

启动前端

```
npm run dev
```

### 后端

进入后端
```
cd back
```
在终端输入如下命令启动后端
```
mvn spring-boot:run
```
